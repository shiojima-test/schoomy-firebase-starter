使い方の流れ
1
Firebase プロジェクトを作成
Firebase コンソールでプロジェクトを作り、Realtime Database を有効化します。
2
firebaseConfig を書き換え
sender.html と receiver.html の firebaseConfig に、自分のプロジェクトの設定値を貼り付けます。
3
sender.html を開いて接続
Chromeで開き、オレンジボードをUSB接続して「接続」ボタンを押します。
4
receiver.html でリアルタイム受信
別のPCやタブで開くと、センサー値がリアルタイムに表示されます。

⚠️ 注意事項

  • Chrome 限定: Web Serial API は Google Chrome（デスクトップ版）でのみ動作します。Safari・Firefox・スマートフォンでは使えません。
  • HTTPS 必須: Web Serial API は HTTPS または localhost でのみ使用できます。GitHub Pages なら自動で HTTPS になります。
  • Firebase 30日ルール: Spark（無料）プランの Realtime Database は30日間アクセスがないと自動停止されます。
  • セキュリティルール: テスト時は読み書き許可でOKですが、公開時は必ず認証ルールを設定してください。
  • ボーレート: オレンジボードのデフォルトは 9600bps です。Arduino IDE 側の設定と合わせてください。
